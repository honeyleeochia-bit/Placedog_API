### Description
A simple image generator that displays **random dog images** based on user-defined width and height.
This project demonstrates working with **image-based APIs** and proper UI handling without JSON fetching.

### API Used
PlaceDog API  
https://placedog.net/

### Features
- Generate random dog images
- Custom width and height input
- Loading indicator while image loads
- Error handling for invalid input or failed image load
- Clean and beginner-friendly UI

### API Details
**Base URL:**  
https://placedog.net/

**Endpoint Pattern:**
/{width}/{height}


### Authentication
- No authentication required for public endpoint
- API key (if used) stored safely in `config.js` and not committed to GitHub

- JavaScript
